✨ Hendyscape

Hendyscape adalah website showcase arsitektur modern yang dirancang dengan tampilan elegan, minimalis, dan interaktif. Website ini menampilkan berbagai layanan, proyek, serta galeri visual dengan pengalaman pengguna yang halus dan menarik.

🛠️ Tech Stack
-HTML5
-Tailwind CSS
-JavaScript
-GSAP (GreenSock Animation Platform)

🚀 Cara Menjalankan Project

Untuk menjalankan project ini dengan benar, silakan jalankan perintah berikut di terminal:
pnpm run watch

Perintah ini digunakan untuk menjalankan proses build dan memantau perubahan file (watch mode), sehingga styling dan tampilan website dapat diperbarui secara otomatis saat melakukan pengembangan.
